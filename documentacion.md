# Estructura de Datos

## Lista Circular Doble

![Lista](src/lista.PNG)

En esta ocacion hemos realizado para nuestro repositorio de Github un ejemplo de como podemos realizar la estructura de una lista Circular Doble o una Lista Doblemente enlazada esto hecho en el leguaje de C++ y en la cual hemos agregado las siguiente funciones para este peque√±o y sencillo ejemplo

### Funciones

* Insertar al inicio 
* Insertar al final
* Elminar un nodo al inicio
* Eliminar un nodo al final
* Obtener el estado actual de la estructura
* Imprension hacia adelante y atras de todos los nodos de la lista
